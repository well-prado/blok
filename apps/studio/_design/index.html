What you're looking at

  • Tabs above swap the iframe between the four screens. Each is a self-contained HTML file you can open standalone too — no build, no server, double-click works.
  • Junior banner at the top of every screen states the assumptions and the open questions for that screen. Push back on those, not on the design wholesale — that's where iteration is cheapest.
  • Real Blok logo sliced from docs/assets/logo/dark.svg (paths 1–3) is embedded inline in every screen — no img src, no fetch.
  • Real run shapes: run_23f1aa6 is the actual cross-runtime-chain run we verified at 14:55 GMT this morning. run_4d8a17b5 is a synthesized C# DEPENDENCY outage — real shape, fictional run id, used to demo the FAILED state.
